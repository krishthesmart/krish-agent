QUICK START (Copy & Paste)
================================================================================

1. Install build tools (ONE TIME):
   pip install --upgrade build twine setuptools wheel


2. Get PyPI API Token:
   - Go to: https://pypi.org/account/register/
   - Create account (if needed)
   - Go to: https://pypi.org/manage/account/
   - Create API token
   - Copy the token (looks like: pypi-AgEIcHlwaS5vcmc...)


3. Configure credentials (Choose ONE):

   OPTION A - Store in ~/.pypirc:
   ---------
   cat > ~/.pypirc << 'EOF'
   [distutils]
   index-servers = pypi

   [pypi]
   repository = https://upload.pypi.org/legacy/
   username = __token__
   password = pypi-YOUR_TOKEN_HERE
   EOF

   chmod 600 ~/.pypirc


   OPTION B - Use environment variables:
   ---------
   export TWINE_USERNAME="__token__"
   export TWINE_PASSWORD="pypi-YOUR_TOKEN_HERE"


4. Publish (from project root):

   bash QUICK_PUBLISH.sh

   OR manually:

   rm -rf build/ dist/ *.egg-info/
   python -m build
   twine upload dist/*


5. Verify installation works:

   pip install krish-agent==3.0.0-GODMODE
   python -c "from krish_agent.godmode import FinalForm; print('✓ GODMODE is live!')"

IF SOMETHING GOES WRONG
================================================================================

1. Invalid token?
   → Regenerate at: https://pypi.org/manage/account/

2. Version already exists?
   → Increment version in setup.py (e.g., 3.0.1-GODMODE)

3. Permission denied?
   → Verify PyPI account has upload rights
   → Try uploading to TestPyPI first: https://test.pypi.org

4. Need to upload again?
   → Change version first, then rebuild and re-upload
